54. Public

Público.

É um modificador de acesso que torna público determinado objeto.

As classes são por padrão public.

Os construtores precisam ser public.

Sendo public, significa que em todo o projeto pode-se acessar os elementos públicos.



55. Private

Privado

Torna o elemento privado à própria classe.

Só dentro da própria classe é possível o acesso.


56. Protected

Protegido.


Por padrão o modificador de acesso é Protected.

O modificador de acesso Protected faz com que o elemento seja visível somente dentro do mesmo pacote
onde o elemento foi declarado.
